## 🚀 Proyecto Destacado

### 🔄 Swapp — Plataforma de Intercambio
Aplicación web que permite realizar intercambios (trueques) de forma segura entre múltiples usuarios.

**Funcionalidades principales:**
- Autenticación y autorización mediante JWT
- Sistema de mensajería entre usuarios
- Integración con la API de Cloudinary para gestión de imágenes de perfil
- Entorno de desarrollo contenerizado con Docker

**Stack utilizado:**  
Flask · SQLAlchemy · React · Bootstrap · Docker
